# Papergrid Industries - Website

A clean, responsive business landing page for **Papergrid Industries**, built with HTML5, CSS3, and Bootstrap 5. This site serves as a product showcase and informational portal for the brand.

## Features
- **Fully Responsive:** Optimized for mobile, tablet, and desktop views using Bootstrap's grid system.
- **Modern UI:** Minimalist black and white theme to reflect the brand's aesthetic.
- **Product Gallery:** A structured showcase of paper products using optimized `.webp` images.
- **Bootstrap Integration:** Utilizes Bootstrap 5 for rapid styling and interactive components (like the mobile navigation toggle).

## Built With
- **HTML5:** Semantic structure.
- **Bootstrap 5.3.8:** CSS Framework for layout and responsiveness.
- **Bootstrap Icons:** For social media and navigation iconography.
- **WebP Images:** For faster loading speeds and better performance.
